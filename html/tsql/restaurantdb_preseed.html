Before a database can be made available to end users a certain amount of information needs to be input. This can be done via an automated process through the application accessing the database or be done manually. In this case I'm showing some manual inputs. Later on I can show examples of automating the process through functions, procedures and packages. This script is not complete, as it is simply a series of INSERT statements.
